# My Personal Blog! 

https://andrew-li-blog.vercel.app/

Forked from [jekyll-klise template](https://github.com/piharpi/jekyll-klise)

## Running

Run local server:

```bash
bundle exec jekyll serve
```

Now running on http://localhost:4000/

To add new post do

```bash
bundle exec jekyll post "Title"
```
